# CoDiagram
A collaborative diagram creation app with real-time editing

## Key Features

- create, resize, move, connect nodes
- change edge types and styles
- add collaborators
- preview where collaborators are on canvas
- undo/redo
- download diagram
- save diagram
- preview all your diagrams
